# Yandex Maps API Next.js Integration

Проект на Next.js с интеграцией Yandex Maps API.

## Технологии

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- ESLint

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Сборка

```bash
npm run build
npm start
```
